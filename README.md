# Discord Contact Form

A powerful WordPress plugin that sends contact form submissions directly to Discord channels via webhooks.

![WordPress Plugin](https://img.shields.io/badge/WordPress-Plugin-blue.svg)
![Version](https://img.shields.io/badge/version-1.0.0-green.svg)
![PHP](https://img.shields.io/badge/PHP-7.4%2B-purple.svg)

##  Features

- **Direct Discord Integration** - Send form data instantly to Discord channels
- **Multilingual Support** - Spanish and English with automatic detection
- **Custom Form Builder** - Create forms with multiple field types
- **Responsive Design** - Mobile-friendly and theme-adaptive
- **AJAX Submissions** - No page reloads, smooth user experience
- **Easy Setup** - Configure webhook URL and start receiving messages
- **Color Customization** - Match your site's design

## Installation

1. **Download** the plugin files
2. **Upload** to `/wp-content/plugins/discord-contact-form/`
3. **Activate** the plugin in WordPress admin
4. **Configure** your Discord webhook URL in **Discord Form** menu

##  Usage

### 1. Set up Discord Webhook
1. Go to your Discord server settings
2. Create a webhook for your channel
3. Copy the webhook URL
4. Paste it in **Discord Form → Configuration**

### 2. Create a Form
1. Go to **Discord Form → Form Editor**
2. Add/remove fields as needed
3. Customize field types and settings
4. Save your form

### 3. Display the Form
Add this shortcode to any page or post:
```
[discord_contact_form]
```
